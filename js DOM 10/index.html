DOM

DOM (Document Object Model) JavaScript ka ek bohot important hissa hai jo aapke HTML code aur browser ke beech ek bridge ka kaam karta hai. Isse aap apni website ko dynamic bana sakte ho(jaise button click par color change hona)html ka sara code hum dom se js ke undar la sakte hai

tree structure me bnata hai jisme sabse pehle window fir document or uske unadar sab kuch html ka html me body h1 p vgra in sab ko node bolte hai (window ,doc html vgra in tree structure)

DOM ke 3 Hisse (The Concept)

1. Document: Aapka pura web page (yani html ka sara code js me ek object me convert ho jata hai jise document bolte hai ).
2.Object: JavaScript ki nazar mein har cheez ek "object" hai. Har tag (div, h1, button) ke paas kuch Properties (jaise color, id) aur Methods (jaise click()) hote hain.
3. Model: Ye ek structure hai jo batata hai kaun kiska parent hai aur kaun kiska bachcha.

jab browser kisi HTML page ko load karta hai, toh vo us HTML ko DOM Tree me convert kar deta hai mtlb html k sare element ko object ya node bana deta hai

DOM manipulation

DOM Manipulation ka matlab hota hai JavaScript se HTML elements ko access karna jis element me change karna hai change, add, delete ya control karna. Simple words me: JS se webpage ko modify karna.


dom manipulation Properties

Properties ka matlab hai aisi DOM properties jinki help se hum element ko access karke uski value check ya change kar sakte hain.
